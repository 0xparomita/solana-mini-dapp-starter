# Solana Mini dApp Starter

An expert-level, flat-structure boilerplate for interacting with the Solana blockchain. This project demonstrates how to connect to a wallet, check balances, and request a Devnet airdrop using a clean, professional interface.

## Features
- **Wallet Integration:** Seamless connection to Solana browser wallets.
- **Real-time Balance:** Fetch and display SOL balances dynamically.
- **Airdrop Tool:** Integrated Devnet faucet functionality for developers.
- **Flat Structure:** All logic contained in the root for maximum clarity.

## How to Run
1. Install dependencies: `npm install`
2. Start the development server: `npm start`
3. Connect your Phantom or Solflare wallet (set to Devnet).

## Technologies
- React.js
- @solana/web3.js
- @solana/wallet-adapter-react
- Tailwind CSS
